Documentation on Managing Links

Support Forums

You can add links here to be displayed on your site, usually using Widgets. By default, links to several sites in the WordPress community are included as examples.

Links may be separated into Link Categories; these are different than the categories used on your posts.

You can customize the display of this screen using the Screen Options tab and/or the dropdown filters above the links table.

If you delete a link, it will be removed permanently, as Links do not have a Trash function yet.
